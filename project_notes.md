## HTML页面分析

### 主要功能模块
1. **个人信息**：姓名、性别、年龄、飞行员编号
2. **飞行员职业特征**：
   - 平均单次飞行时长
   - 年度飞行时间
   - 机型选择
   - 高空巡航时间占比
   - 每月跨时区飞行次数
   - 驾驶舱温度中位数
   - 应急备降经历次数
   - 航空餐盐分摄入等级

3. **代谢指标体系**：
   - 血清草酸盐
   - 超敏C反应蛋白
   - 脂蛋白(a)水平
   - 钙磷代谢指标（钙、磷、钙磷比）
   - 尿液指标（尿液比重、尿常规PH值）
   - 肾功能指标（肌酐、尿素、尿素/肌酐比率）
   - 尿酸相关指标
   - 其他代谢指标（ALT、空腹血糖、糖化血红蛋白、甲状旁腺素、降钙素、胆固醇、HDL、LDL、甘油三酯）

4. **综合健康指标**：
   - 体脂率
   - 身体测量指标（身高、体重、腰围、臀围、血压）
   - 深静脉血栓史
   - 飞行中补水频率
   - 辐射暴露程度
   - 每日饮水量
   - 每周运动时间
   - 工作压力指数
   - 睡眠质量指数
   - 心理状态指数

### 风险计算算法
- 飞行风险 = 职业特征归一化值的平均
- 代谢风险 = 代谢指标归一化值的加权平均
- 综合健康风险 = 健康指标归一化值的平均
- 总体风险 = (飞行风险 + 代谢风险 + 综合健康风险) / 3

### 风险等级
- 低风险：< 20%
- 中等风险：20-50%
- 高风险：50-80%
- 极高风险：> 80%

## 系统需求
1. 响应式前端界面（支持手机和电脑）
2. 后台数据库存储飞行员数据
3. 数据导出功能（一键下载）
4. 远端API连接功能
5. 用户认证系统
