开发文档

快速上手 PilotStone 的完整指南

快速开始

三步启动PilotStone本地开发环境

# 1. 克隆项目
git clone https://github.com/PilotStone - 飞行员尿石症AI风险评估系统/pilot-stone-prediction.git
cd pilot-stone-prediction

# 2. 安装依赖
pnpm install

# 3. 启动开发服务器
pnpm run dev

# 构建GitHub Pages静态版本
BUILD_MODE=static pnpm build

项目结构

模块化架构设计

pilot-stone-prediction/
├── client/                    # 前端应用 (React 19 + TypeScript)
│   ├── src/
│   │   ├── pages/            # 页面组件
│   │   │   ├── Assessment.tsx # 风险评估页面
│   │   │   ├── History.tsx    # 历史记录页面
│   │   │   ├── Papers.tsx     # 研究论文页面
│   │   │   └── Research.tsx   # 研究数据页面
│   │   ├── components/       # UI组件
│   │   │   ├── MetabolicForm.tsx    # 代谢指标表单
│   │   │   ├── AssessmentResult.tsx # 评估结果展示
│   │   │   └── PrintReport.tsx      # PDF报告生成
│   │   └── lib/              # 工具库
│   └── index.html
├── server/                    # 后端服务 (Node.js + Express)
│   ├── riskCalculator.ts     # 核心风险计算引擎
│   ├── reportGenerator.ts    # 报告生成器
│   ├── seerResearchService.ts # SEER研究数据服务
│   └── db.ts                 # 数据库操作
├── shared/                    # 共享代码
│   ├── assessment.ts         # 类型定义
│   ├── defaults.ts           # 默认值配置
│   └── suggestionEngine.ts   # 建议生成引擎
├── docs/                      # GitHub Pages构建输出
├── drizzle/                   # 数据库Schema
└── scripts/                   # 工具脚本

API参考

核心API接口说明

POST /api/assessment
提交风险评估数据，计算三维风险分数
参数类型说明
genderstring性别 (male/female)
flightDurationnumber平均单次飞行时长 (小时)
annualFlightHoursnumber年度飞行小时数
metabolicDataobject代谢指标数据
healthDataobject健康指标数据
GET /api/papers
获取研究论文列表
参数类型说明
keywordstring搜索关键词
tagstring标签筛选
yearnumber年份筛选
pagenumber页码
POST /api/report
生成PDF评估报告
参数类型说明
assessmentIdstring评估记录ID
formatstring报告格式 (pdf/html)

部署方案

多种部署方式可选

GitHub Pages (推荐)

静态部署，免费托管，适合Demo和文档站点

# 构建静态版本
BUILD_MODE=static pnpm build
# 推送到docs/目录
git add docs/ && git commit -m "deploy"
git push origin main

Docker部署

容器化部署，适合生产环境

# 构建镜像
docker-compose build
# 启动服务
docker-compose up -d

Vercel部署

Serverless部署，自动扩展

# 安装Vercel CLI
npm i -g vercel
# 部署
vercel --prod
