关于 PilotStone

守护飞行员健康，保障飞行安全

项目愿景

飞行员因高空环境、脱水、饮食特殊性，尿石症发病率显著高于普通人群。PilotStone 致力于通过AI技术，为飞行员提供精准的尿石症风险评估和个性化健康管理方案。

我们的系统基于国家发明专利（专利号：2024116733167），整合职业特征、代谢指标和综合健康三维度数据，通过机器学习模型实现精准预测。

PilotStone 是 PilotStone - 飞行员尿石症AI风险评估系统 OPC 生态系统的核心项目之一，完全开源，社区驱动，持续迭代。

技术架构

现代化全栈技术栈

前端技术

React 19 + TypeScript 5.9 + Tailwind CSS + Vite 7

UI组件: shadcn/ui + Radix UI

后端技术

Node.js + Express + tRPC

API: RESTful + Type-safe RPC

数据存储

MySQL (生产) / localStorage (静态模式)

ORM: Drizzle + Schema-first

数据可视化

Recharts + 交互式图表

支持: 雷达图 折线图 柱状图

部署方案

GitHub Pages / Vercel / Docker

CI/CD: GitHub Actions

安全合规

端到端加密，GDPR/HIPAA合规

标准: 医疗数据安全

项目结构
