风险评估算法

飞行风险计算

归一化各项飞行指标，加权平均计算飞行风险分数：

flightRisk = (normFlightDuration + normAnnualFlight + normAltitude + normTimezone + normCockpitTemp + normDiversions + normSalt) / 7

代谢风险计算

综合分析20+项代谢指标，包括草酸盐、尿酸结晶指数、钙磷比、尿液比重等核心指标：

metabolicRisk = weightedSum(serumOxalate, uricAcidCrystal, calciumRatio, urineGravity, ureaCreatinine, lipoproteinA, hsCRP, ...) / totalWeight

健康风险计算

评估体脂率、血压、运动量、饮水量、睡眠质量等生活方式指标：

healthRisk = weightedSum(bodyFat, bloodPressure, exercise, waterIntake, sleepQuality) / totalWeight

综合风险等级判定

三维加权综合评估，四级风险等级划分：

overallRisk = flightRisk × 0.3 + metabolicRisk × 0.4 + healthRisk × 0.3

低风险(<20%) | 中等风险(20-50%) | 高风险(50-80%) | 极高风险(>80%)
